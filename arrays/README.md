| 0 | Juice |
| ---- | --- | 
| 1 | Apple |
| 2 | Cheese |

| Operation | Complexity |
| ------ | -------- |
| Lookup | 1 |
| Push | 1 |
| insert | O(n) |
| delete | O(n) |


# Static Arrays - Fixed in Length

| Operation | Complexity |
| ------ | -------- |
| Lookup | 1 |
| Push* | 1 |       
| insert | O(n) |
| delete | O(n) |

\* Can be O(n), if the push goes beyond the array length
